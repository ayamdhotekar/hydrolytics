# Hydrolytics

Hydrolytics is a small project focused on analyzing water usage and quality data. The idea is to explore how data can be used to better understand water systems and detect potential issues early.

---

## What it does

* Loads and processes water usage data
* Computes basic statistics (average, min, max)
* Detects unusual spikes in usage (possible leaks)
* Predicts future usage using simple forecasting

---

## Why I built this

I wanted to work on something related to **real-world data problems**, especially around sustainability. Water management is an important area, and this project is a way to experiment with data analysis in that space.
